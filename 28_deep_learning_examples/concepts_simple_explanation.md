# Simple Deep Learning Concepts

## 1. Neural Network / ANN
A neural network is used mainly for tabular data, numerical data, and general prediction tasks.

Example in this ZIP:
Customer churn prediction.

## 2. CNN
CNN means Convolutional Neural Network.
It is mostly used for image data.

Example in this ZIP:
Circle vs square image classification.

## 3. RNN / LSTM
RNN means Recurrent Neural Network.
It is used for sequence data like text, time series, and speech.

Example in this ZIP:
Bank customer review sentiment classification.
